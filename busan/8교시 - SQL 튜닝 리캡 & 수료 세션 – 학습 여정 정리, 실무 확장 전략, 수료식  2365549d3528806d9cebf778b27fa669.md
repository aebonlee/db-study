### 2. [실무 적용을 위한 핵심 전략 정리 – 15분]

| 영역 | 전략 핵심 요약 |
| --- | --- |
| 조건절 | 함수 제거, IN/EXISTS 전환, =/BETWEEN 적극 활용 |
| 인덱스 | 선행컬럼 구조 설계, Function-Based Index, 중복 제거 |
| 실행계획 | FILTER vs ACCESS, COST 집중지점 확인 |
| 힌트 | INDEX, FULL, LEADING, USE_HASH 등 목적별 제한 사용 |
| 통계/히스토그램 | NDV, 분포 왜곡 판단 → 히스토그램 수집 |
| 병렬/파티션 | 대용량 처리 시 PQ, RANGE 파티션 조합 |
| 락/대기 | TX WAIT, LATCH WAIT 진단 및 트랜잭션 크기 조정 |
| 보고서화 | SQL_ID, 실행계획 전후 비교, 개선율 정량화 필수 |

### 3. [추가 학습 및 확장 방향 안내 – 10분]

### ✅ 학습 연계:

- [SQLP] 자격 준비: AWR 해석, 파라미터, 옵티마이저 Hint 적용
- [PL/SQL 튜닝] 연계 과정: 루프 구조, 커서 FOR LOOP, BULK
- [DBA 과정] 연계: Library Cache, PGA/SGA 구조, 파라미터 관리
- [ETL 성능 튜닝] 연계: 병렬 처리, Partition Pruning, Sort Merge 조정

### 📚 추천 참고자료:

- Oracle 공식 문서: [Execution Plan](https://docs.oracle.com/en/database/)
- 실전 튜닝 사례 블로그, SQLD 시험 기출 모음
- AWR/ASH 기반 실무 진단 시뮬레이션
